# EvaPersepcionAccion - Simulador de Navegación de EVA

Este proyecto es una aplicación desarrollada en **Java SE** que simula el comportamiento de percepción y acción de un agente robótico. En la simulación, el personaje de **EVA** (de la película WALL-E) debe navegar a través de un laberinto con el objetivo de encontrar la salida de forma autónoma.

## Tecnologías Utilizadas

* **Lenguaje:** [Java](https://www.oracle.com/java/) (Java SE)
* **Gráficos:** Java Standard Edition (Swing/AWT)
* **Entorno de desarrollo sugerido:** Eclipse o NetBeans.

## Estructura del Repositorio

* `/src`: Contiene el código fuente (`.java`) con la lógica del agente, el renderizado del mapa y la gestión del laberinto.
* `/bin` o `/out`: (Si existe) Archivos compilados `.class`.
* `/resources`: Imágenes de EVA y texturas del laberinto.

## Instalación y Ejecución

1. **Clonar el repositorio:**
   ```bash
   git clone [https://github.com/garridoandreag/EvaPersepcionAccion.git](https://github.com/garridoandreag/EvaPersepcionAccion.git)
   ```
2. Compilar: Si usas la terminal, navega a la carpeta src y compila la clase principal:
Bash
javac Main.java

3. Ejecutar:
Bash
java Main
